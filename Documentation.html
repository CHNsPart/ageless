Gym Videos Site Documentation

Welcome to the documentation for the Gym Videos Site template. This guide will help you set up, customize, and use the template effectively.

Setup

To set up the project, follow these steps:

  1. Ensure you have Node.js and npm installed on your machine.
  2. Download or clone the repository to your local machine.
  3. Navigate to the project directory and run the following command to install the dependencies:
    4. npm install
  4. Start the development server using the following command:
    5. npm run dev
  5. Open your browser and navigate to http://localhost:3000 to view the site.

Usage

The main components of the site are located in the src/components directory. Each component is a separate file that can be edited individually.

Customization

To customize the template, you can modify the following files:

  • src/components/About.tsx: Update the About section content.
  • src/components/Hero.tsx: Modify the Hero section.
  • src/components/Footer.tsx: Edit the footer content and links.

You can also add new components or modify existing ones as per your requirements.
